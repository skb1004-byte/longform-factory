================================================================================
API 엔드포인트 요약
================================================================================

GET /health
  └─ 헬스 체크 (status, timestamp, api_key_configured)

GET /voices
  └─ ElevenLabs 사용 가능 음성 목록

POST /tts
  ├─ text: 1-5000자 텍스트 (필수)
  ├─ voice_preset: korean_male | korean_female
  ├─ voice_id: 커스텀 음성 ID
  ├─ model_id: eleven_multilingual_v2 (기본)
  ├─ stability: 0.0-1.0 (기본: 0.5)
  ├─ similarity_boost: 0.0-1.0 (기본: 0.75)
  ├─ output_format: mp3_44100_128 (기본)
  └─ filename: 저장 파일명 (자동 생성 가능)

POST /tts/batch
  ├─ items: TTS 요청 배열 (1-100개)
  └─ Response: total, success_count, results[]

빠른 시작 (3단계)
================================================================================

1. 환경 설정
   $ cp .env.example .env
   # .env 파일에 ELEVENLABS_API_KEY 입력

2. 빌드
   $ docker-compose build

3. 실행
   $ docker-compose up -d

4. 테스트
   $ python test_client.py

또는:
   $ curl http://localhost:8001/health
